# Atlas

Strava route organiser.

## Initial feature list

* Tagging
* Folders
* Sorting by
  - name
  - distance
  - elevation
  - elevation/distance
  - creation date
  - estimated completion time
  - tags
* Filtering by
  - distance
  - elevation
  - elevation/distance
  - estimated completion time
  - tags
* Search using all of the above

## Possible future features

* Route comparison
